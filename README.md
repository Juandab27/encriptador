# 🔐 Encriptador de Texto

Aplicación web desarrollada como parte del curso de formación en programación de Alura LATAM.

## 📖 Descripción

El **Encriptador de Texto** permite cifrar y descifrar mensajes utilizando reglas de sustitución específicas.

El usuario puede ingresar un texto, encriptarlo para proteger su contenido y posteriormente desencriptarlo para recuperar el mensaje original.

Es una aplicación enfocada en reforzar conceptos de lógica de programación, manipulación de strings y desarrollo web básico.

---

## 🚀 Funcionalidades

* 🔒 Encriptar texto
* 🔓 Desencriptar texto
* 📋 Copiar resultado al portapapeles
